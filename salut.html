Balises Paires

  • head: Balise en téte de page, titre de la page caractére de la page..
  • body : Corp de la page
  • h1-h6 : Titre/sous titre..
  • em : Mettre en valeur, (style italique)
  • strong : Texte en gras
  • mark : Texte surligné par défaut en jaune
  • ul : liste à puce non numéroté

  • li : Puce

  • a href="dossier/lien">phrase suivante ou nom du lien

Italique, gras souligné...

En italique avec ( font-style ):
  • italic : le texte sera en italique
  • oblique : le texte sera légèrement penché
  • normal : le texte sera normal(par defaut), cela permet d'annuler une mise en "italique" avec em
En Gras avec ( font-weight ) :
  • bold : le texte sera en gras
  • normal : le texte sera normal(par defaut) cela permet d'annuler une mise en "gras" avec strong
Soulignement et autres décorations avec ( text-decoration ) :
  • underline : souligné
  • line-through : barré
  • overline : ligne au dessus
  • nome : normal

L'lignement avec ( text-align ) :

Fonctionne qu'avec les balises de type "block"
  • lest : le texte sera aligné à gauche (par défaut)
  • center : le texte sera centrer
  • right : le texte sera aligné à droite
  • justify : le texte sera "justifier", il permet de prendre toute le largeur sans laisser d'espace blanc

Les flottants avec ( float ) :

Permet de faire flotter un element autour d'un texte.
On peut l'utiliser avec les balises "inline" et "block"
  • left : l'élément flottera à gauche
  • right : l'élément flottera à droite
Ex code html :
<p><img src="flash.gif" class="imageflottante" alt="Image flottante" /> Ceci est un texte normal de paragraphe, écrit à la suite de l'image et qui l'habillera car l'image est flottante.</p>

Puis ex code CSS :
.imageflottante
{
float: left;
}

Stopper un flottant avec ( clear ) :

  • left : le texte se poursuit en dessous aprés un float: left;
  • right : le texte se poursuit en dessous aprés un float: right;
  • both : le texte se poursuit en dessous, que ce soit aprés un float: left; ou aprés un float: right;
